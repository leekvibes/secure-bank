# Shot List

## Core Screens
- Dashboard overview
- New secure link page
- Client-facing secure form
- Submission success state
- Link detail timeline
- Submission reveal view

## Recording Specs
- Vertical: 1080x1920 (Reels/TikTok)
- Horizontal: 1920x1080 (X/LinkedIn)
- Keep clips 3-6s each
- Use cursor highlights for clicks

## Sequence (45s)
1. Open dashboard
2. Create link
3. Copy/send link
4. Open client form
5. Submit sample data
6. Return to dashboard submitted state
7. Reveal/confirm flow
