# Content Pack v1

This pack is ready for immediate social posting and outbound DM.

## Included
- `screenshots/` 6 reusable product screenshots
- `captions/social-posts.md` 10 ready-to-post social captions
- `dm/outreach-templates.md` 3 DM scripts + follow-ups
- `videos/reel-scripts.md` 3 short-form video scripts (30-45s)
- `videos/shot-list.md` shot plan for recording polished demos
- `scripts/capture-playwright.mjs` auto-capture scaffold (requires Playwright install)

## Quick Start (Today)
1. Post screenshot `01-dashboard-overview.png` with Post #1.
2. Send 20 DMs using Template A.
3. Post screenshot `02-secure-form-view.png` with Post #2.
4. Record Reel Script #1 using `videos/shot-list.md`.
